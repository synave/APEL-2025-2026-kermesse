# APEL-2025-2026-kermesse
Circulaire pour la friterie lors de la kermesse dans le cadre de l'APEL et cartes de jeux 5 et 10 parties.

# Outils

Ordinateur sous Windows 11.

L'affiche a été créée avec Word de la suite office 365.

Le fond de l'affiche a été créé par chatGPT et modifié par gemini.

Famille de police utilisée : "Bodoni MT"

Les cartes de jeux ont été créés avec gimp.

Mes feuilles d'assets ont été créées avec chatgpt puis assembler pour créer les cartes

La police utilisée pour les cartes de jeux est "Luckiest guy" (voir credits).

# Pour modifier l'affiche

Ouvrir le fichier source (APEL-2025-2026-kermesse.docx) avec Word. Les polices sont standard dans word.

Ouvrir les fichiers APEL-2025-2026-cartes_5_jeux.xcf (ou 10) avec gimp. La police utilisée doit être installée. (voir credits)

# Suppléments

Une version image PNG et une version PDF sont également fournies pour l'affiche.
